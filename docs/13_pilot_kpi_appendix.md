# 13. KPI-приложение к пилотному контракту

## 1) Общие принципы

- KPI фиксируются до старта пилота
- Baseline рассчитывается на согласованном историческом окне
- Расчет эффекта выполняется еженедельно и подтверждается обеими сторонами
- Спорные кейсы выносятся в joint review board

## 2) KPI по NPT Predictor

### KPI-1: Снижение NPT

- Формула: NPT Reduction (%) = (NPT_baseline - NPT_pilot) / NPT_baseline * 100%
- Единица: %
- Цель: >= 10%
- Источник: буровые отчеты, события NPT

### KPI-2: Precision алертов по NPT

- Формула: Precision = TP / (TP + FP)
- Единица: доля
- Цель: >= 0.70
- Источник: журнал алертов + верификация инженером

### KPI-3: Adoption рекомендаций

- Формула: Adoption (%) = Executed Recommendations / Total Recommended * 100%
- Единица: %
- Цель: >= 60%

## 3) KPI по ESP Failure Predictor

### KPI-4: Снижение аварийных отказов ЭЦН

- Формула: Failure Reduction (%) = (Failures_baseline - Failures_pilot) / Failures_baseline * 100%
- Единица: %
- Цель: >= 15%
- Источник: CMMS + акты ремонтов

### KPI-5: Recall критических отказов

- Формула: Recall = TP / (TP + FN)
- Единица: доля
- Цель: >= 0.75

### KPI-6: Время раннего предупреждения

- Формула: Early Warning Time = t_failure - t_alert
- Единица: часы
- Цель: медиана >= 24 ч

## 4) Финансовый KPI

### KPI-7: Подтвержденный экономический эффект

- Формула: Verified Value = Value_NPT + Value_ESP - Additional OPEX
- Единица: [валюта]
- Подтверждение: joint sign-off ежемесячно

## 5) Правила учета данных

- Пропуски данных более [X]% периода отмечаются как data quality event
- Периоды планового капремонта исключаются из сравнения
- Экстремальные форс-мажорные события анализируются отдельно

## 6) Границы ответственности

- KPI зависят от фактического исполнения рекомендаций операционной командой
- При низком adoption (< [порог]) итоговый эффект интерпретируется с оговоркой

## 7) Процедура согласования итогов

1. Предварительный weekly KPI report от Исполнителя
2. Проверка заказчиком в течение [N] рабочих дней
3. Подписание consolidated monthly KPI statement
4. Итоговый pilot KPI closure report на 90-й день
