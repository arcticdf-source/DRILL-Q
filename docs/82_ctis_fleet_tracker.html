CTIS Fleet Tracker

Это отдельный рабочий инструмент по логике центральной инженерной сводки. Он не хранит реальные названия и цифры, а дает ежедневную форму по проектам: план/факт за сутки, месяц и год, текущий забой, состав работ, stage, статус смены и краткий operational note. Сводка заполняется руками каждый день, пересчитывает агрегаты и формирует fleet-layer для диспетчера, руководителя бурения и remote ops.

How To Use This Like Real CTIS

  • Одна строка = один активный проект/буровая в текущем daily fleet view.
  • Сюда вносится только operational minimum: где мы, что сделали за сутки, против какого плана идем и что нужно дожать.
  • Если нужно, под эту строку открывается уже deeper layer: ГТИ, супервайзерская сводка, action desk.

What This Tool Solves

  • Не дает fleet-контексту расползтись по Excel-файлам и чатам.
  • Делает ежедневный срез сопоставимым между проектами.
  • Отделяет уровень диспетчерской картины от детального GTI / supervisor level.
