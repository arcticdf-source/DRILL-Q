Периметр, Роли И Deliverables

Scope
Scope
  • Актив / месторождение
  • Количество скважин
  • Историческое окно данных
  • Бурение и/или добыча
Responsibilities
  • Исполнитель: модели, SLA, value reporting
  • Заказчик: данные, SME, операционное внедрение
Deliverables
  • Интеграционный дизайн
  • 2 AI-модели
  • Дашборды и журнал рекомендаций
  • Weekly value-reports
  • Pilot closure report

SLA, HSE И Приемка

Acceptance
  • Platform availability: 99.5%
  • Critical incident response: до 30 минут
  • Weekly report: не позднее согласованного дня
  • AI не нарушает safety-барьеры
  • Критические решения только после подтверждения заказчиком
  • Успех пилота: достигнуты KPI, подтвержден эффект, есть applicability sign-off
