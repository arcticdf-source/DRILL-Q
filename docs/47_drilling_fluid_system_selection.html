Пакеты химической природы (без брендов)

Глинистый WBM (спуд/кондуктор)

Бентонит + щелочной буфер + минимальный контроль фильтрации. Низкая стоимость, высокая операционная простота.

Полимер-глинистый ингибированный WBM

Соли K/Na/Ca + полимер-вязкостник (ксантан/целлюлозный) + фильтрационный агент (крахмал/полимер) + лубрикант + LCM при необходимости.

Synthetic/NAF класс

Инвертная/синтетическая база + эмульсионный пакет + утяжелитель + контроль фильтрации и реологии для сложных интервалов и узких окон.

Хранилище техданных (для AI-агентов)

База профилей месторождений хранится в JSON-структуре workspace: knowledge/field_profiles_index.json, knowledge/field_profiles/*.json, knowledge/fluid_system_templates.json, knowledge/similarity_rules.json. Это позволяет агентам быстро находить аналогичные месторождения и использовать типовые семейства растворов как стартовую точку.
