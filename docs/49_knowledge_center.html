Центр базы знаний
Сервер: проверка... На главную
Статус: ожидание обновления.

Как пользоваться

1. Загружайте документы в папки knowledge/inbox или просто в knowledge/inbox/dropzone.
2. Нажимайте кнопку обновления базы знаний.
3. Индексатор пересоберет каталог и поисковые чанки.
4. Страница AI Инженер начнет использовать новые знания.
5. Для месторождений/скважин можно строить baseline-профили через knowledge/tools/build_profile_from_sources.py.
Если браузер не запускает .cmd напрямую, используйте один из вариантов:
knowledge/tools/update_knowledge_index.cmd
knowledge/tools/update_knowledge_index.ps1
или запустите Python-скрипт вручную:
python knowledge/tools/build_knowledge_index.py
Создание baseline-профиля из документов:
python knowledge/tools/build_profile_from_sources.py profile_id path/to/file1 path/to/file2

Ограничения

Из статической HTML-страницы нельзя надежно и безопасно запускать локальный Python полностью автоматически во всех браузерах. Поэтому кнопка ведет к локальному скрипту обновления. После запуска индекс обновляется автоматически.
