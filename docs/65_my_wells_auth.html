Вход администратора

Это отдельный защищенный аккаунт владельца. Он не связан с обычной регистрацией пользователей и используется только для полного админ-доступа к каталогу и выдачи прав партнерам.

Админ-доступ разрешен только для отдельного admin-аккаунта.

Регистрация

Делитесь логином и паролем с командой, чтобы иметь общий доступ.

Как это работает

Регистрация команды Создайте общий логин и пароль для команды, чтобы работать в едином контуре скважин по месторождению.
Защищённый доступ Только после входа можно открывать страницу «Мои скважины» и добавлять свои карточки скважин.
Свой дашборд Каждая созданная скважина открывается в отдельном дашборде со структурой модулей. Пока блоки на этой странице только обзорные и не кликаются.
