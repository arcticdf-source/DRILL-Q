График выполнения работ

Инфографический модуль по программе бурения ВЧНГКМ на 2027–2028 годы. Все сводные показатели на странице собраны по исходному файлу «Приложение 15. Ориентировочный график выполнения работ.xlsx»: 119 скважин, 4 лота, проходка, цикл, этап ДПМ, буровые отходы и ритм запуска по месяцам. Переключатель года пересчитывает карточки и лоты без ручной правки.

Что показывает страница

Здесь не просто список строк из Excel, а нормальная операционная панель: масштабы программы, волна запуска по месяцам, раскладка по лотам, профиль по годам и логика расчетов, заложенных в исходном графике. Это даёт быстрый вход в программу работ до ухода в детальную таблицу.

Логика расчета

Используются исходные значения графика: этап бурения считается по датам начала и окончания, цикл складывается из ДПМ и бурения, а объем буровых отходов равен сумме БШ, БР и БСВ. На странице агрегаты суммируются по всем строкам файла или по выбранному году.

2027 vs 2028

Сравнение двух лет по числу скважин, проходке, суммарному циклу и буровым отходам. Это быстро показывает, где фактическая нагрузка выше и где концентрируется основной объем программы.

Тепловая карта запусков по месяцам

Чем выше число в ячейке, тем плотнее волна стартов в соответствующий месяц. Для полного контура показаны все 24 месяца программы, для фильтра по году остаются только месяцы выбранного периода.

Профиль буровых отходов

Общий объем отходов формируется из трех компонентов: БШ, БР и БСВ. В исходном графике средний объем на одну скважину стабилен, поэтому здесь особенно важна общая сумма по программе и по выбранному периоду.

Как считаются показатели

проверка логики

Этап бурения

В исходном Excel длительность бурения считается от даты начала до даты окончания включительно. Это логика столбца «Этап бурения, сут».

Этап бурения = Дата окончания - Дата начала + 1

Цикл скважины

Полный цикл программы по строке складывается из этапа ДПМ и бурения. На странице суммарный цикл считается как сумма этих значений по выбранному набору строк.

Цикл = Этап ДПМ + Этап бурения

Буровые отходы

Итог по отходам равен сумме компонент БШ, БР и БСВ. Для программы в целом на странице показаны как полный объем, так и разложение по трем компонентам.

Всего отходов = БШ + БР + БСВ
