Раздел 1. Общие данные

Сводка релевантных общих требований по тендеру «Южно-Обская» для формирования программы буровых растворов и инженерно-технологического сопровождения.

Источники: ГТЗ 1/2 · ТЗ Услуга БР · Приложение 6.1

Ключевые проектные вводные

Объект: строительство поисково-оценочных скважин №1 и №2 «Южно-Обская» (вариант СПБУ).
Район строительства: Обская губа, Южно-Обский лицензионный участок.
Заказчик/проектный контур: ООО «Газпром нефть шельф».
Цель бурения: поиск залежей углеводородов.
Проектный горизонт: верхняя юра (баженовская свита).
Тип скважин: поисково-оценочные, профиль - вертикальный.
Проектные глубины: скв. №1 - 2630 м; скв. №2 - 2660 м (по вертикали от стола ротора).
Глубина воды в точке бурения: 5.5 м; альтитуда стола ротора: 30 м (предварительно).
Норматив по раствору: параметры и контроль по API 13B-1 / ISO 10414-1:2008.
Интервалы БР по ТЗ: 0-120, 120-800, 800-1935, 1935-2645 м с переходом от морской воды к ингибированным калий-хлор-полимерным системам.
