# 17. Бурение — Этап 3: Масштабирование

## Цель

Тиражировать пилотные практики на дополнительные бригады, скважины и активы с сохранением качества решений.

## Периметр

- 3-6 буровых бригад
- 20-60 скважин/интервалов
- Период: 12-24 недели

## Ключевые задачи

1. Стандартизация playbook по бурению
2. Развертывание единого каталога алертов и рекомендаций
3. Усиление MLOps: контроль drift, автоматические retrain-пайплайны
4. Формирование регулярных операционных review по value realization

## KPI этапа

- NPT Reduction 12-18%
- Стабильность модели на новых площадках
- Доля автоматизированных рекомендаций >= 70%
- SLA соблюдается >= 99.5%

## Критерий завершения

- Repeatable delivery модель подтверждена
- Экономический эффект устойчив на расширенном периметре
- Команда заказчика готова к масштабной эксплуатации
