Раздел 4. Буровые растворы

Модуль собирает mud-program layer по ВЧНГ так же, как в полевом проекте по Уренгою: с разбивкой по интервалам, рабочими параметрами раствора, объемами приготовления, номенклатурой химреагентов, кольматационными материалами и специальными требованиями к сервису. Ниже развернуты три разных сценария, потому что для каждого типа скважины архитектура БР отличается по числу стадий, системам БС и материалоемкости.

3типа программ БР
18стадий и интервалов
5систем БР и заканчиваний
645-805 м³приготовление по типу

Что внутри страницы

Цветной интервальный разрез по каждому типу скважины с привязкой к системам БР.
Профиль плотностей, объемов приготовления и переходов между стадиями.
Паспорт каждого интервала: тип раствора, насосы, МСП, диаметр долота, кавернозность, параметры и служебные ограничения.
Матрица химреагентов и отдельный блок спецтребований по кольматантам, смазке, KCl, гидрооснове и очистке.

Фактура источника

Основа страницы собрана из таблиц по параметрам БР для трех типов скважин и номенклатурных требований к химреагентам и материалам. Там, где приложение задает не дозировку, а только обязательную номенклатуру, это отражено как матрица обязательного наличия, а не как выдуманный расход.

На этой странице объемы приготовления и интервальные параметры взяты из таблиц. Для химреагентов добавлен инженерный ориентир по стартовым концентрациям в кг/м³: это рабочие диапазоны для стартовой рецептуры и лабораторной доводки, а не подмена тендерной таблицы.
