Логика ввода

Все значения вводятся в метрической системе. Для модуля Юнга можно вводить обычное число, например 10.3, или строку вида 10.3e6.

Длина трещины

Обычно брать 0,1524 м. Для узкого near-wall сценария можно быстро переключиться на 0,5 диаметра ствола.

Статус расчета
