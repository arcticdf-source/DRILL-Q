# 06. Дорожная карта запуска (180 дней)

## Day 0–30: Foundation

- Зарегистрировать юрлицо, лицензии, базовый комплаенс
- Нанять core team (CEO, CTO, CDAO, VP Ops)
- Выбрать 1–2 целевых вертикали (например, добыча + надежность)
- Заключить 2–3 MoU с потенциальными пилотными клиентами
- Развернуть базовый data platform контур

Критерий выхода:

- Есть pipeline пилотов и подписанный дизайн-партнер

## Day 31–90: MVP + Pilot

- Подключить первые источники данных (SCADA/CMMS/ERP)
- Запустить 2 AI-кейса на реальных данных
- Настроить command center и mobile field app
- Зафиксировать baseline KPI и методику расчета эффекта
- Подписать коммерческий пилотный контракт

Критерий выхода:

- Есть подтвержденный эффект минимум по 1 KPI

## Day 91–180: Scale Readiness

- Стандартизировать playbooks внедрения
- Запустить MLOps промышленного уровня
- Построить customer success функцию
- Подготовить security/audit пакет для enterprise клиентов
- Сформировать 12-месячный план продаж

Критерий выхода:

- 3+ оплачиваемых клиента, repeatable delivery модель

## Контрольные метрики на 180 день

- 3 пилота, из них 2 конвертированы в долгосрочные контракты
- ARR run-rate >= 1.2M USD
- Средний time-to-value <= 45 дней
- Первая библиотека отраслевых моделей в production
