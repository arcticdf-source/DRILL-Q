Пилотные Кейсы

Case pack
01

NPT Predictor

Прямая связь с потерями времени и стоимости бурения. Высокая частота событий позволяет быстро обучать и калибровать модель на ранних интервалах пилота.

  • KPI: NPT -10-15%
  • Lead time: 30-120 минут
  • Precision: >= 0.70
  • Adoption: >= 60%
02

ESP Failure Predictor

Высокая стоимость аварийных остановок и подъемов/спусков. Доступная телеметрия делает кейс удобным для value-based monetization.

  • Failure reduction: 15-25%
  • Early warning: 24-72 часа
  • Recall: >= 0.75
  • Downtime: -10-20%
