Доступ партнеров

Здесь админ выдает доступ логинам, которым разрешено менять только оперативные поля: остатки на складах и цены. Название продукта, группа и статус страницы остаются под контролем владельца.

Текущая версия работает на localStorage браузера. Позже можно заменить это backend-хранилищем без смены структуры страниц.
