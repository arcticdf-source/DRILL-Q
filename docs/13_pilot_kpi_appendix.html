Метрики По Категориям

Formula pack
NPT
  • NPT Reduction (%) = (NPT_baseline - NPT_pilot) / NPT_baseline * 100%
  • Precision = TP / (TP + FP)
  • Adoption (%) = Executed Recommendations / Total Recommended * 100%
ESP
  • Failure Reduction (%) = (Failures_baseline - Failures_pilot) / Failures_baseline * 100%
  • Recall = TP / (TP + FN)
  • Early Warning Time = t_failure - t_alert
Finance
  • Verified Value = Value_NPT + Value_ESP - Additional OPEX
  • Joint sign-off ежемесячно
  • Спорные кейсы идут в joint review board

Цели, Источники И Правила Учета

Control rules

NPT Goals

  • NPT reduction >= 10%
  • Precision >= 0.70
  • Adoption >= 60%

ESP Goals

  • Failure reduction >= 15%
  • Recall >= 0.75
  • Median early warning >= 24h

Data Rules

  • Пропуски > [X]% = data quality event
  • Плановый капремонт исключается
  • Force majeure анализируется отдельно

Closure Process

  • Weekly KPI report
  • Проверка заказчиком
  • Monthly KPI statement
  • 90-day closure report
