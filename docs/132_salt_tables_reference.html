Что смотреть первым

Если задача на chloride brine, сначала выбирай таблицу по типу соли, затем сверяй плотность и water activity, после чего проверяй crystallization point в °C на ожидаемую температуру участка.

Как пользоваться в полях

Для оперативной оценки удобно искать по целевой плотности в кг/л или кг/м³, а затем брать рядом ppm Cl⁻, загрузку соли в кг/м³ и water fraction без дополнительного ручного пересчета.

Ограничение справки

Таблицы даны как reference screen для проектирования и контроля; финальную рабочую программу все равно нужно сверять по чистоте соли, температуре и лабораторным данным конкретного реагента.

Основные поля таблиц

Weight % показывает массовую долю соли, density выводится в кг/л и кг/м³, а chloride / cation columns помогают быстро перейти к lab control и retort / titration интерпретации.

Для solids screen

Эти reference tables удобно использовать рядом с [Cl⁻], brine density, salt volume correction и выбором солевого base fluid при отдельном solids-анализе.

Критическая проверка

Если требуемая плотность попадает между строками, для поля лучше брать ближайшую более консервативную комбинацию по crystallization point и затем уточнять лабораторно.
