## Что я сделаю сразу после появления файлов в workspace

1. Прочитаю ТЗ и приложения, а не только названия файлов.
2. Соберу точную матрицу `источник -> раздел -> блок данных`.
3. Определю, какие разделы можно сразу заполнить полностью, а какие придется оставить как черновики.
4. По аналогии с Уренгоем начну сборку отдельных HTML-страниц для нового проекта.
